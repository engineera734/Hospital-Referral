هذه الحزمة مبنية فوق المرحلة السابقة، ولا تحتاج SQL جديد إذا كنت قد نفذت SQL الخاص بالمراحل السابقة بنجاح.

أهم التعديلات:
- التقرير العام يوضح عدد الحالات المنتظرة والمستقبلة لكل طبيب.
- التقرير المحدد خاص بالطبيب/الفترة المحددة فقط، ولا يعرض جداول موظفي الاستقبال والمحاسبة.
- تصفية المستحقات في الإدارة والمحاسبة تدعم: كل الأشهر أو فترة شهرية محددة.
- إضافة زر مستقل لتصفية "المصروف مسبقًا" للطبيب إذا كان لديه رصيد مصروف سابق.
- إزالة تبويب البحث من صفحة الاستقبال والإبقاء على قائمة الاستقبال فقط.
- تنزيل المرفقات يتم مباشرة على الجهاز.

إذا كنت قد نفذت SQL السابق بالفعل، لا تعيد تنفيذ ملفات SQL مرة أخرى إلا بعد المراجعة.
